## üõ†Ô∏è Setup

### 1. Clone the repository

```bash
git clone https://github.com/Realtrickz/dc-rich-presence
cd discord-24-7-rich-presence
```

### 2. Install dependencies
```bash
npm install
```

### 3. Configure environment variables

Rename `.env.example` ‚Üí `.env` and add your Discord token:
```env
TOKEN=XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

### 4. Configure Rich Presence
Edit config.yml and customize your presence:
```yml
details: "Coding"
state: "Building something cool"
largeImage: "logo"
smallImage: "online"
startTimestamp: true
```

### 5. Start the script
```bash
npm run start
```
